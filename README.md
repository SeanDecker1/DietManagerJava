# TERM-PROJECT: Diet Manager

An online Diet Manager application to be developed in Java 8 

## Team

- Josh Schrader
- Kevin Gleason

## Prerequisites

- Java 8

- IntelliJ - as developer platform


## How to run

1. Clone the repository and go to the root directory on your local machine.

2. Enter the src directory

3. run in your terminal: javac ./driver/DietManager.java

4. run in your terminal: java driver/DietManager

5. Follow the instructions provided by the application
